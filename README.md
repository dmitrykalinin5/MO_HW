# Домашние задания по методам оптимизации

Проект содержит решение ДЗ1.1, ДЗ1.2, ДЗ2.1, ДЗ2.2 и ДЗ2.3 для варианта 9.

## Как запустить все сразу

```bash
python3 homework.py
```

Этот файл по очереди запускает все отдельные программы и собирает общую сводку в `build/results.json`.

## Python-файлы

- `dz1_1_cubic_approximation.py` — ДЗ1.1. Считает минимум функции одной переменной методом кубической аппроксимации.
- `dz1_2_visualization.py` — ДЗ1.2. Строит графики исходной функции и кубических аппроксимаций.
- `dz2_1_contours.py` — ДЗ2.1. Считает траектории покоординатного, градиентного и наискорейшего спуска, строит линии уровня и ломаные приближений.
- `dz2_2_newton.py` — ДЗ2.2. Находит стационарные точки аналитически, классифицирует их и запускает метод Ньютона.
- `dz2_3_object_model.py` — ДЗ2.3. Содержит простую объектную модель трех методов оптимизации и строит ее рисунок.
- `homework.py` — общий запускатель всех программ.

## Про `Contours and solutions.ipynb`

Для ДЗ2.1 использована схема из `Contours and solutions.ipynb`: строится сетка `meshgrid`, затем линии уровня через `contour`, затем поверх них рисуется ломаная последовательности приближений.

Важная доработка под задание: в notebook линии уровня задавались как обычное число уровней, а в `dz2_1_contours.py` уровни берутся из значений функции в вершинах траектории. Поэтому каждая линия уровня проходит через соответствующую вершину ломаной.
